# OOPMaster - Object-Oriented Programming in PHP

This project demonstrates key Object-Oriented Programming (OOP) concepts using PHP, showcasing the use of classes, inheritance, encapsulation, and more. It includes examples of magic methods, static methods, namespaces, and type hinting, among other OOP principles.

## Project Structure

- **classes/**: Contains the class files organized by autoloading.

### Key Files:

1. **class-autoload.php**
   - Demonstrates class autoloading with namespaces.

2. **constructor-and-destructor.php**
   - Example of how to use constructors and destructors in PHP OOP.

3. **encapsulation.php**
   - Demonstrates encapsulation by using private properties and public methods.

4. **inheritence.php**
   - Shows how inheritance works in PHP, allowing one class to inherit properties and methods from another.

5. **inheritence-deep.php**
   - Delves deeper into inheritance, with examples of multi-level inheritance and method overriding.

6. **instanceof.php**
   - Demonstrates the `instanceof` keyword, used to check if an object is an instance of a particular class.

7. **magic-method-constant.php**
   - Example of using PHP magic methods and constants for dynamic method handling.

8. **method-override.php**
   - Demonstrates method overriding in derived classes.

9. **namespace.php**
   - Example of how to organize classes using namespaces for better code structure.

10. **object-and-class.php**
    - Basic example of creating and using objects and classes in PHP.

11. **properties-and-method.php**
    - Shows how to define properties and methods, and how they can be used within classes.

12. **static-properties-method.php**
    - Demonstrates the use of static properties and static methods in PHP.

13. **tostring.php**
    - Example of using the `__toString()` magic method for representing an object as a string.

14. **type-hinting.php**
    - Demonstrates type hinting in method parameters and return types.

## How to Run the Project

### Prerequisites:
- PHP 7.4 or above
- [Herd](https://herdphp.com) or any other local PHP server.

### Instructions:

1. Clone the repository:
    ```bash
    git clone https://your-repo-url.git
    cd OOPMaster
    ```

2. Start the PHP server using Herd:
    ```bash
    herd link
    herd secure
    ```

3. Access the project in the browser:
    ```
    http://OOPMaster.test
    ```
